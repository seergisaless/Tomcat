Apache Tomcat

Apache Tomcat es una implementación de código abierto de las tecnologías Java Servlet, JavaServer Pages, Java Expression Language y Java WebSocket. Estas tecnologías están desarrolladas mediante el proceso de la Comunidad Java (JCP).

El software Apache Tomcat se desarrolla en un entorno abierto y participativo bajo la licencia Apache 2. El proyecto está diseñado para ser un punto de encuentro para colaboradores de todas partes del mundo.

Apache Tomcat impulsa numerosas aplicaciones web de misión crítica en una gran variedad de industrias.
